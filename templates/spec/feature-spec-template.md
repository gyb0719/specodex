# 기능 명세 개요

## 목적
- 주된 사용자 가치와 해결하려는 문제를 한 문단으로 요약하세요.

## 사용자 스토리
- [ ] 역할, 행위, 결과 형식으로 작성합니다.
- [ ] 미정 사항은 `[CLARIFY: 세부 질문]`으로 표시합니다.

## 요구 사항
- 기능 요구, 비기능 요구를 bullet로 나누어 기술합니다.
- 수용 기준은 “주어진 상황/언제/그러면” 형식으로 작성합니다.

## 검토 체크리스트
- [ ] 모든 `[CLARIFY: …]` 항목이 해소되었나요?
- [ ] 사용자 여정이 누락되지 않았나요?
- [ ] 성능·보안 등의 제약이 명시되었나요?
